# 🏆 Système de Finale Ultra Personnalisable

## 📋 Vue d'Ensemble

Le système de finale a été complètement refait pour offrir une personnalisation maximale et des jokers vraiment cool et stratégiques !

---

## 🎮 5 Modes de Finale

### 1. ❤️ **Système de Vies**
- Chaque équipe commence avec X vies (1-10 configurable)
- Une mauvaise réponse = -1 vie
- 0 vie = élimination
- **Idéal pour** : Finales longues avec tension progressive

### 2. 💀 **Élimination**
- Chaque mauvaise réponse élimine directement
- Le dernier debout gagne
- **Idéal pour** : Finales rapides et intenses

### 3. 🎯 **Course aux Points**
- Premier à atteindre X points (50-500) gagne
- Points configurables par question
- **Idéal pour** : Finales basées sur les connaissances

### 4. ⚡ **Mort Subite**
- Une seule vie pour tous
- Une erreur = fin du jeu
- **Idéal pour** : Finales ultra tendues de 5 minutes

### 5. 🏆 **Tournoi**
- Bracket 1v1 jusqu'au champion
- Élimination directe par match
- **Idéal pour** : Grandes finales avec beaucoup d'équipes

## ✨ 15 Jokers Cool et Stratégiques

### 🛡️ Jokers Défensifs

#### **BOUCLIER** 🛡️ (Rare)
- **Effet** : Protège contre une mauvaise réponse
- **Utilisation** : Pas de pénalité ni perte de vie
- **Stratégie** : À garder pour les questions difficiles

#### **RESURRECTION** ❤️‍🩹 (Légendaire)
- **Effet** : Récupère une vie perdue
- **Utilisation** : Mode vies uniquement
- **Stratégie** : Permet de revenir dans la partie

---

### ⚔️ Jokers Offensifs

#### **VOL DE POINTS** 🦹 (Épique)
- **Effet** : Vole 10 points à l'équipe de ton choix
- **Utilisation** : Cibler le leader pour égaliser
- **Stratégie** : Bloque les adversaires proches de la victoire

#### **SABOTAGE** ⚡ (Épique)
- **Effet** : Bloque le buzzer d'une équipe pour cette question
- **Utilisation** : Empêche un adversaire de répondre
- **Stratégie** : Neutralise les équipes rapides au buzzer

#### **CONFUSION** 🌪️ (Rare)
- **Effet** : Inverse 2 options de réponse pour les autres équipes
- **Utilisation** : QCM uniquement - sème le chaos
- **Stratégie** : Perturbe les adversaires sur des questions faciles

---

### 👁️ Jokers Informatifs

#### **VISION** 👁️ (Légendaire)
- **Effet** : Révèle quelle option est correcte
- **Utilisation** : QCM uniquement - tu vois la bonne réponse
- **Stratégie** : Points garantis sur question difficile

#### **INDICE** 💡 (Commun)
- **Effet** : Élimine 2 mauvaises réponses
- **Utilisation** : QCM - réduit les options de 4 à 2
- **Stratégie** : Augmente tes chances de 25% à 50%

#### **TEMPS LIBRE** ⏰ (Rare)
- **Effet** : Ajoute 15 secondes au chrono
- **Utilisation** : Pour les questions complexes
- **Stratégie** : Plus de temps pour réfléchir

---

### 🎲 Jokers Tactiques

#### **DOUBLE OU RIEN** 🎲 (Épique)
- **Effet** : Double les points si bonne réponse, 0 si mauvaise
- **Utilisation** : Quand tu es sûr de toi
- **Stratégie** : Gros risque, grosse récompense

#### **PASSE TON TOUR** ⏭️ (Commun)
- **Effet** : Saute cette question sans pénalité
- **Utilisation** : Quand la question est impossible
- **Stratégie** : Évite de perdre une vie

#### **MULTIPLICATEUR** ✖️2 (Épique)
- **Effet** : X2 sur les points de cette question
- **Utilisation** : Questions à gros points
- **Stratégie** : Maximise tes gains

#### **CHANCE** 🍀 (Rare)
- **Effet** : Réponse aléatoire - 25% de réussir
- **Utilisation** : Quand t'as vraiment aucune idée
- **Stratégie** : Hail Mary, dernière chance

---

### 🤝 Jokers d'Équipe

#### **PARTAGE** 🤝 (Rare)
- **Effet** : Partage 50% de tes points avec une autre équipe
- **Utilisation** : Aide un ami à remonter
- **Stratégie** : Forme des coalitions

#### **ALLIANCE** 🔗 (Légendaire)
- **Effet** : Forme une alliance - points partagés pendant 3 questions
- **Utilisation** : Alliance stratégique
- **Stratégie** : Deux équipes ensemble plus fortes

#### **TRAHISON** 🗡️ (Légendaire)
- **Effet** : Rompt une alliance et vole TOUS les points de l'allié
- **Utilisation** : Trahis ton alliance au dernier moment
- **Stratégie** : Le move le plus fourbe du jeu

| Paramètre | Min | Max | Par Défaut | Description |
|-----------|-----|-----|------------|-------------|
| Nb Finalistes | 2 | 16 | 8 | Nombre d'équipes en finale |

### Options Avancées

- **Autoriser Comebacks** : Les équipes éliminées peuvent revenir avec joker spécial
- **Recharge Jokers** : Les jokers se rechargent toutes les 5 questions
- **Config par Joker** : 0-10 de chaque joker par finaliste

## 💡 Conseils Pro

### Pour les Animateurs

1. **Équilibrage** :
   - Mode Vies (3 vies) : 6-8 jokers par équipe
   - Mode Élimination : 4-5 jokers (risque élevé)
   - Course aux Points : 7-10 jokers (longue durée)
   - Mort Subite : 3-4 jokers max (rapide)

2. **Mix Rareté** :
   - 40% Communs
   - 30% Rares
   - 20% Épiques
   - 10% Légendaires

3. **Tension** :
   - Moins de jokers = plus de tension
   - Plus de jokers = plus de spectacle

### Pour les Joueurs

1. **Gestion** :
   - Ne gaspille pas les jokers dès le début
   - Garde les légendaires pour la fin
   - Utilise les communs en début de partie

2. **Timing** :
   - VISION sur questions à gros points
   - BOUCLIER quand dernière vie
   - SABOTAGE sur l'équipe en tête

3. **Alliances** :
   - Forme des alliances tôt
   - Trahis au bon moment (dernier tiers)
   - Méfie-toi des autres !

## 🚀 Utilisation en Régie

### Étape 1 : Configuration
1. Va dans Mode Final
2. Choisis le mode (Vies, Élimination, etc.)
3. Configure nb finalistes, vies, points
4. Sélectionne quantité de chaque joker
5. Active options avancées si besoin

### Étape 2 : Création
1. Clique "Créer la Finale"
2. Les N meilleures équipes sont sélectionnées
3. Les jokers sont distribués
4. Les vies sont initialisées (si mode Vies)

### Étape 3 : Lancement
1. "Lancer l'Introduction" → Écran d'intro épique
2. "Activer la Finale" → La partie commence
3. Les finalistes peuvent utiliser leurs jokers depuis leur client

### Étape 4 : Gestion
- **Mode Vies** : Tu vois les vies en temps réel, peux retirer manuellement
- **Course aux Points** : Barre de progression vers objectif
- **Tracking jokers** : Historique d'utilisation accessible
